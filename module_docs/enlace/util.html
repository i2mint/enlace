enlace.util.is_skippable(name: str) → bool[source]

Return True if a directory name should be skipped during discovery.

Directories starting with ‘_’ or ‘.’ are skipped.

>>> is_skippable("_internal")
True
>>> is_skippable(".git")
True
>>> is_skippable("my_app")
False
