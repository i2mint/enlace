enlace.diagnose

Diagnose app compatibility with enlace.

Scans an app directory for patterns that would prevent or complicate mounting under the enlace multi-app platform. Reports issues at three severity levels (CRITICAL, MEDIUM, LOW) with specific file locations, explanations, and suggested fixes.

Usage:

from enlace.diagnose import diagnose_app
report = diagnose_app("/path/to/my_app")
print(report)

Or from the CLI:

enlace diagnose /path/to/my_app
enlace diagnose /path/to/my_app --json
